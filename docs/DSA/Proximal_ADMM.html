Proximal Gradient and ADMM

Proximal Gradient

一般的非线性优化问题主要是通过Gradient Descent方法来解决的。但是这必须要求目标函数是可导的，但是实际上，我们经常会碰到一些目标函数不可导的优化问题（如LASSO,分位数回归等）。Proximal Gradient (以下简称PG) 算法主要是为了解决一类不可导无约束优化问题（大部分的有约束优化问题也可以通过 Lagriange 方法转化为无约束优化问题。）

我们考虑如下的优化问题： minxF(x)=f(x)+g(x) \min_{x} F(x)=f(x)+g(x) ​x​min​​F(x)=f(x)+g(x) 这里我们要求f(x)f(x)f(x)是可导的，凸的，并且是 β\betaβ-Smooth的，但是对于g(x)g(x)g(x)我们只要求 g(x)g(x)g(x)是凸的（其实很多的非凸问题也可以用Proximal Gradient Descent 来解决，不过理论证明会相对比较复杂，所以这里我们假设函数都是凸函数）。

这种问题有很多，例如
